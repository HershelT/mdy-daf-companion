# Release And Marketing Plan

## Positioning

MDY Daf Companion is a Claude Code plugin for developers who want their agentic coding sessions to become steady Daf Yomi time.

Short pitch:

> Let Claude Code work while you keep up with the Daf. MDY Daf Companion plays Rabbi Eli Stefansky's latest MDY shiur, pauses when Claude needs you, saves your place, and tracks your learning progress.

## Audience

- Developers who learn Daf Yomi.
- MDY listeners who use AI coding tools.
- Busy professionals who want background Torah learning during focused work.
- Communities that want to encourage consistent daily learning.

## Product Name Options

- MDY Daf Companion
- Daf While Claude Codes
- YomiCode Companion
- MDY Workbench
- DafFlow for Claude Code

Recommended: `MDY Daf Companion` if permission/brand posture is clear. Otherwise use `Daf Companion for Claude Code` and describe MDY support without implying official affiliation.

## Brand And Legal Posture

Before public launch:

- Add clear disclaimer: "Not affiliated with or endorsed by Mercaz Daf Yomi unless a partnership is established."
- Link users to official MDY sites and YouTube channel.
- Do not use MDY logos without permission.
- Do not sell access to MDY videos.
- Consider contacting MDY for permission, feedback, or partnership.

## Free Vs Paid

Free:

- Current Daf playback.
- Claude lifecycle pause/resume.
- Progress saving.
- Basic stats.
- Local privacy-first storage.

Supporter or Pro:

- Advanced dashboard.
- Multi-device sync.
- Calendar catch-up planner.
- Exportable reports.
- Community groups.
- Custom goals.
- Dedications/sponsor rotation.

Tone:

- "Support ongoing development" is a better fit than hard gating.
- Keep the Torah-learning core available.

## Marketplace Listing Draft

Title:

MDY Daf Companion

Short description:

Play the latest Rabbi Eli Stefansky Daf Yomi shiur while Claude Code works, with automatic pause/resume and local learning stats.

Long description:

MDY Daf Companion turns Claude Code sessions into steady Daf Yomi time. When Claude starts working, the plugin opens the right MDY shiur for today's daf. When Claude finishes, asks for permission, or waits for your next prompt, the shiur pauses and your progress is saved. Track watched minutes, coding minutes, completed dafim, streaks, and catch-up progress, all stored locally by default.

Highlights:

- Latest MDY Daf Yomi shiur resolution.
- English and Hebrew format preferences.
- Automatic play/pause based on Claude Code lifecycle.
- Local progress and stats.
- Status line and slash commands.
- Shabbos/Yom Tov guard.
- Privacy-first design.

Disclaimer:

This plugin is an independent companion and is not affiliated with or endorsed by Mercaz Daf Yomi unless otherwise stated.
