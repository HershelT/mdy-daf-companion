---
description: Configure MDY Daf Companion preferences.
---

Help the user choose language, format, timezone, guard behavior, and auto-open behavior. Use `mdy-daf setup` with explicit flags such as `--language english --format full --timezone America/Chicago --guard true --auto-open true`. If `mdy-daf` is not available on PATH, run `node "${CLAUDE_PLUGIN_ROOT}/dist/src/cli.js" setup` with the same flags.
