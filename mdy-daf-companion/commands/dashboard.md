---
description: Open the MDY Daf Companion Stats view inside the Electron companion.
---

Run `mdy-daf open-dashboard`. If `mdy-daf` is not available on PATH, run `node "${CLAUDE_PLUGIN_ROOT}/dist/src/cli.js" open-dashboard`. Explain that it opens the same floating Electron companion directly to the Stats view. Do not open a regular browser window automatically.
