# Analyze Daf Stats

Use this skill when the user asks about learning progress, coding-versus-watching time, streaks, or catch-up plans.

Guidelines:

1. Use local stats from the plugin daemon or database.
2. Explain stats in encouraging, practical language.
3. Do not shame missed days.
4. Do not expose raw project paths, prompt text, source code, or transcript content.
5. When data is missing, say what needs to be tracked before the stat can be calculated.
