# Support

## Getting Help

Start with:

```bash
mdy-daf doctor
```

Then check:

- [Install And Compatibility](docs/install-and-compatibility.md)
- [Privacy](mdy-daf-companion/PRIVACY.md)
- [Progress Ledger](docs/progress-ledger.md)

## Common Issues

- Desktop remote/cloud sessions do not support plugins.
- SSH/dev-container sessions may need port forwarding for the player.
- If `dist/src/cli.js` is missing in development, run `npm run build`.
- If resolver results look wrong, run `mdy-daf resolve --date YYYY-MM-DD`.
